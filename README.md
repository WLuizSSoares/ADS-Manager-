# 🎓 ADS Manager - Gestor Acadêmico

Sistema de gerenciamento de membros acadêmicos desenvolvido para praticar os pilares da **Programação Orientada a Objetos (POO)** em Java.

## 🚀 Funcionalidades
- **Cadastro de Alunos:** Validação de nome, idade e matrícula.
- **Cadastro de Professores:** Registro de disciplina lecionada.
- **Listagem Dinâmica:** Exibição de todos os membros cadastrados no sistema.
- **Entrada Blindada:** Tratamento de exceções para garantir que o sistema não trave com entradas inválidas (letras em campos numéricos, números negativos ou campos vazios).

## 🛠️ Tecnologias e Conceitos Aplicados
- **Linguagem:** Java 17+ (ou a versão que você usou).
- **POO:** Herança, Polimorfismo e Encapsulamento.
- **Tratamento de Exceções:** Uso de `try-catch` e lançamento de `IllegalArgumentException` para validações personalizadas.
- **Arquitetura:** Organização em pacotes (`model`, `service`, `util`).
